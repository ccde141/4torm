# 季风 Chat · 对话

季风是 4torm 最基础的用法:你和一个 Agent 一对一聊天。你提需求,它一边思考一边动手——查资料、读写文件、跑命令,做完把结果告诉你。

和普通 AI 聊天工具最大的不同是:这里的 Agent 是你**事先配好的一个"人"**(有自己的提示词、模型、工具和工作区),不是一个临时的对话窗口。同一个 Agent 今天在这里陪你聊,明天可以被拉进工作流或定时任务里干活。

左侧切到 **季风 · 对话**,顶部选一个 Agent,就能开始。没有 Agent 就先去控制台建一个,见[快速开始](../guide/getting-started)。

![季风对话界面](/screenshots/季风-Subagent调用1.png)

## 看它"怎么想的"

发出消息后,Agent 不是直接回你一句话,而是"想一想 → 调工具 → 看结果 → 再决定下一步"这样推进。这个过程会实时显示,中间步骤折叠成卡片,不想看细节就只看最后的回答。

用带推理能力的模型时,它的思考过程单独显示在一张卡片里,落定后自动收起。

## 回复途中继续发消息

Agent 还在忙的时候你可以继续打字,这时发送**不会打断它**,而是排队——等它这一轮做完再处理你的新消息。排队的消息显示在输入框上方,点一下可取消。适合想到什么先记下来。

## 让 Agent 自己拆活:子任务委托

遇到大任务,Agent 会把一部分工作**派给临时助手**并行处理,自己继续主线。比如"分析整个项目并写报告",它可能派几个助手分头看不同目录,各自返回摘要再汇总。

要不要拆、怎么拆由 Agent 自己判断,你不用操作。界面上会显示"子任务进行中",完成后结果自动并回主对话。

![子任务进行中](/screenshots/季风-Subagent调用2.png)

## 聊太久了:压缩

对话太长会占用大量上下文,又慢又贵。发送 `/compact` 可以把前面的对话浓缩成摘要,保留关键信息、丢掉琐碎过程,让对话轻装继续。压缩过的部分会标成"压缩归档"。

## 随时换模型

顶部模型下拉框对话中途也能换。简单问答用快的,复杂推理换强的,按需切换。
